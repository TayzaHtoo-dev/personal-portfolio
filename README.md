# Personal Portfolio

## Overview
This portfolio showcases the work and projects of Tayza Htoo Dev, aiming to highlight skills, experiences, and technical achievements. It serves as a comprehensive presentation of my capabilities in the field of web development.

## Sections
- **About Me:** A brief introduction and professional background.
- **Projects:** Highlights of key projects with descriptions and links.
- **Skills:** An overview of technical skills and proficiencies.
- **Testimonials:** Feedback from colleagues and clients.
- **Contact:** Information on how to reach me for inquiries or collaborations.

## Features
- **Responsive Design:** Optimized for both desktop and mobile devices.
- **Interactive Elements:** Engaging features like animations and transitions.
- **Search Functionality:** Easily find projects or sections of interest.
- **Dark/Light Theme Switch:** Personalize the viewing experience according to preference.

## How to Use
1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/TayzaHtoo-dev/personal-portfolio.git
   ```
2. Open `index.html` in your browser to view the portfolio.
3. Navigate through the sections using the menu.

## Customization Instructions
- **Changing Content:** Edit the `.html` files in the `src` directory to update project descriptions and skills.
- **Styling:** Customize styles in the `style.css` file to match personal branding.
- **Adding New Projects:** Add new project sections by duplicating existing sections in the HTML files and updating the links.
